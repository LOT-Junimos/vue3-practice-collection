# HorizontalScrolling - Vue3 水平滚动照片展示组件
> 一个基于 Vue3 + GSAP + Lenis 实现的水平滚动照片墙组件，支持粘性定位滚动触发、平滑滚动体验、响应式尺寸适配

## 🌐 在线预览
https://vue3-practice-collection-hs.vercel.app/

## 🎯 功能亮点
1. 水平滚动核心：基于 GSAP ScrollTrigger 实现滚动进度联动，滚动时照片墙水平平移
2. Lenis 平滑滚动：替代原生滚动，实现丝滑的滚动体验，easing 曲线优化过渡效果
3. 粘性定位布局：滚动过程中照片容器固定在视口顶部，仅内容水平移动，视觉聚焦性强
4. 响应式适配：窗口尺寸变化时自动重新计算滚动距离和容器高度，适配不同屏幕分辨率
5. 滚动状态管理：监听滚动离开/返回状态，确保窗口缩放后位置不偏移
6. 内存管理优化：组件卸载时清除 ScrollTrigger 实例/resize 监听，避免内存泄漏
7. 布局细节设计：照片卡片固定宽高比，间距均匀，背景色对比增强视觉层次

## 📱 效果说明
- 默认状态：页面上下为占位区块，中间紫色背景容器固定视口高度，照片卡片横向排列超出屏幕
- 滚动触发：向下滚动时，照片墙随滚动进度水平向左移动，卡片依次展示
- 粘性定位：滚动过程中照片容器始终固定在视口顶部，仅内容横向滑动
- 窗口缩放：调整浏览器窗口大小时，自动重新计算滚动总距离和容器高度，位置自动校准
- 滚动离开：滚动至容器外时，照片墙固定在最右侧；返回时恢复滚动联动
- 响应式适配：不同屏幕宽度下，照片卡片尺寸/间距比例保持一致，核心交互逻辑不变

## 🛠 技术点（Vue3 + GSAP + Lenis 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Script Setup 语法糖   | Vue3 推荐写法，代码简洁无冗余，无需手动导出 |
| ref 响应式引用        | 管理DOM元素/滚动状态参数，状态联动统一 |
| GSAP ScrollTrigger    | 监听滚动进度，实现滚动与动画的精准联动，替代原生 scroll 事件 |
| Lenis 平滑滚动库      | 实现高性能平滑滚动，自定义 easing 曲线，提升滚动体验 |
| 响应式尺寸计算        | 监听 resize 事件，动态计算滚动距离和容器高度，适配不同屏幕 |
| 粘性定位 CSS          | position: sticky 实现容器固定，overflow: hidden 隐藏超出内容 |
| 生命周期管理          | onMounted 初始化/onUnmounted 清理动画/监听，避免内存泄漏 |
| CSS 弹性布局          | flex 实现照片卡片横向排列，flex-shrink: 0 固定卡片尺寸 |
| scoped 样式隔离       | 组件样式独立，避免全局样式污染 |
