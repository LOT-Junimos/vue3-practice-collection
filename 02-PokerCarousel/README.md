# Vue3 扑克牌轮播组件
> 一个基于 Vue3 Composition API 实现的扑克牌堆叠轮播组件，点击顶部扑克牌触发轮播切换，模拟真实扑克牌递出的交互效果，支持无限循环和图片预加载优化。

## 🌐 在线预览
https://vue3-practice-collection-poker-carousel.vercel.app/ 

## 🎯 功能亮点
1. 堆叠布局设计：自定义旋转角度+偏移量，模拟扑克牌真实堆叠效果，旋转中心贴合物理逻辑
2. 智能动画控制：移动过程带过渡动画，重置位置时无动画，避免视觉断层
3. 图片预加载：初始化时预加载所有图片，解决首次加载空白问题，提升交互体验
4. 无限循环轮播：图片索引自动取模，实现无间断轮播，支持任意数量图片适配
5. 响应式适配：采用 rem/vmin 单位，兼容不同尺寸屏幕，布局不塌陷
6. 样式隔离：scoped 样式封装，避免全局样式污染，可直接嵌入任意 Vue3 项目

## 📱 效果说明
- 初始状态：5张扑克牌以不同旋转角度堆叠展示，层级分明，贴合物理堆叠逻辑
- 点击交互：点击顶部透明触发层，所有扑克牌依次向对应位置移动（带0.3s过渡动画）
- 轮播逻辑：最底部扑克牌自动替换为新图片，无动画重置到顶部位置，实现无限轮播
- 视觉体验：旋转中心为左下角，模拟真实扑克牌递出动作，交互反馈自然

## 🛠 技术点（Vue3 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Composition API       | setup 语法糖 + ref 响应式，逻辑更聚合 |
| onMounted 生命周期    | 组件挂载后初始化图片和布局，保证DOM就绪 |
| 动态样式绑定 (:style) | 动态控制 transform/zIndex/transition，实现动画与布局联动 |
| 数组映射 (map)        | 极简逻辑实现扑克牌位置更新，代码易维护 |
| 图片预加载 (new Image) | 提前加载所有图片资源，优化加载体验     |
| CSS transform-origin  | 自定义旋转中心，实现真实物理堆叠效果   |
| 取模运算 (%)          | 图片索引循环复用，实现无限轮播逻辑     |
| scoped 样式           | 组件样式隔离，transform/transition 组合实现丝滑动效 |
