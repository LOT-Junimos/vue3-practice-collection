# Vue3 全屏侧边菜单组件
> 一个基于 Vue3 Setup 语法糖实现的全屏菜单组件，点击右上角汉堡按钮可展开/收起全屏菜单，包含交互动效和hover样式优化。

## 🎯 功能亮点
1. 响应式状态管理：用 Vue3 ref 实现菜单/按钮状态联动
2. 交互动效：菜单展开/收起带过渡动画，文字hover有下划线动效
3. 样式封装：scoped 样式隔离，避免全局样式污染
4. 状态联动：按钮和菜单共用一个 isActive 变量，状态同步无冗余

## 📱 效果说明
- 默认状态：右上角显示橙色汉堡按钮，菜单隐藏
- 点击按钮：按钮切换为关闭图标，全屏紫色菜单从无到有展开（带缩放+透明度过渡）
- 菜单文字hover：文字下方出现橙色下划线（从右到左展开）
- 再次点击：菜单收起，按钮恢复为汉堡图标

## 🛠 技术点（Vue3 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Script Setup 语法糖   | Vue3 推荐的简洁写法，无需return |
| ref 响应式变量        | 控制菜单/按钮的激活状态        |
| 动态class绑定         | 根据 isActive 切换样式类      |
| 点击事件绑定 (@click) | 实现菜单切换逻辑              |
| scoped 样式           | 组件样式隔离，避免冲突        |
| CSS 过渡动画          | 菜单展开/收起的渐变+缩放效果   |
