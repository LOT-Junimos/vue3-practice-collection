# MagneticBall - Vue3 磁吸小球交互组件
> 一个基于 Vue3 + GSAP 实现的磁吸小球交互组件，鼠标移动时触发小球弹性偏移与连线动画，支持响应式布局

## 🌐 在线预览
https://vue3-practice-collection-mb.vercel.app/

## 🎯 功能亮点
1. 磁吸交互核心：自定义磁力强度系数控制小球偏移距离，支持正负值调整磁吸方向，交互更灵活
2. GSAP 动画驱动：使用 GSAP 时间轴精准控制小球/连线的弹性动画，ease 曲线优化过渡效果，丝滑无卡顿
3. SVG 图形优化：紫色小球 + 半透明白色连线，深色背景增强视觉层次，线条透明度可配置
4. 重复创建防护：初始化时清空旧元素，避免多次调用生成重复小球/线条
5. 响应式尺寸适配：兼容 SVG 原生宽高属性和容器计算宽高，适配不同屏幕大小
6. 动画队列管理：鼠标移动时终止未完成动画，避免动画堆积，保证交互即时性
7. 内存管理强化：组件卸载时清空 SVG 容器 + 终止所有 GSAP 动画，彻底避免内存泄漏

## 📱 效果说明
- 默认状态：SVG 容器内均匀排列紫色小球，每个小球中心有白色圆点，小球与原位有半透明白色连线
- 鼠标触发：鼠标移动至小球感应半径内时，小球按磁力系数向鼠标方向偏移，连线跟随拉伸
- 弹性恢复：偏移动画执行后延迟 0.1 秒触发恢复动画，小球平滑回归初始位置，连线同步复位
- 磁力可控：通过 `magneticStrength` 系数调整偏移强度，正负值可实现「吸引/排斥」两种效果
- 响应式适配：调整浏览器窗口大小时，SVG 容器尺寸自适应，小球布局自动重新计算
- 性能优化：快速移动鼠标时，未完成动画会被立即终止，优先响应最新鼠标位置

## 🛠 技术点（Vue3 + GSAP + SVG 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Script Setup 语法糖   | Vue3 推荐写法，代码简洁无冗余，无需手动导出 |
| ref 响应式引用        | 管理 SVG 容器 DOM 元素/磁吸状态参数，状态联动统一 |
| GSAP Timeline 时间轴  | 串联偏移/恢复动画，精准控制动画顺序与延迟，kill() 终止旧动画避免堆积 |
| SVG 元素动态创建      | 使用 createElementNS 动态生成 circle/line 元素，自定义填充/描边/透明度 |
| 鼠标位置精准计算      | 基于容器坐标偏移计算相对位置，结合勾股定理判断感应范围，磁吸交互精准 |
| 磁力强度系数          | 自定义系数控制偏移距离，支持吸引（负值）/排斥（正值）两种交互模式 |
| 重复创建防护          | 初始化时清空容器旧元素，避免多次调用生成重复图形 |
| 生命周期强化管理      | onUnmounted 清理鼠标监听 + 终止 GSAP 动画 + 清空 SVG 容器，彻底避免内存泄漏 |
| CSS 定位优化          | 父容器 relative 定位，SVG absolute 居中，transform-origin 确保坐标计算准确 |
| scoped 样式隔离       | 组件样式独立，SVG 样式精准控制，避免全局样式污染 |
