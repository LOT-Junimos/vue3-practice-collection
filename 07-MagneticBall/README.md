# MagneticBall - Vue3 磁吸小球交互组件
> 一个基于 Vue3 + GSAP 实现的磁吸小球交互组件，鼠标移动时触发小球弹性偏移与连线动画，支持响应式布局

## 🌐 在线预览
https://vue3-practice-collection-mb.vercel.app/

## 🎯 功能亮点
1. 磁吸交互核心：基于鼠标位置计算距离，实现小球磁吸偏移效果，超出指定半径自动恢复原位
2. GSAP 动画驱动：使用 GSAP 时间轴控制小球/连线的弹性动画，ease 曲线优化过渡效果，丝滑无卡顿
3. SVG 图形绘制：纯 SVG 实现小球、中心点、连线的绘制，矢量图形适配任意分辨率无模糊
4. 响应式布局：监听容器尺寸变化，自动重新计算小球初始位置，适配不同屏幕大小
5. 动画队列管理：鼠标移动时终止未完成动画，避免动画堆积，保证交互即时性
6. 内存管理优化：组件卸载时清除鼠标监听事件，GSAP 动画实例自动销毁，避免内存泄漏
7. 视觉层次设计：绿色小球 + 白色中心点 + 白色连线，深色背景增强视觉对比，交互反馈清晰

## 📱 效果说明
- 默认状态：SVG 容器内均匀排列绿色小球，每个小球中心有白色圆点，小球与原位有白色连线
- 鼠标触发：鼠标移动至小球磁吸半径内时，小球向鼠标方向偏移，连线跟随拉伸
- 弹性恢复：偏移动画执行后自动触发恢复动画，小球平滑回归初始位置，连线同步复位
- 即时响应：快速移动鼠标时，未完成的动画会被终止，立即响应最新鼠标位置
- 响应式适配：调整浏览器窗口大小时，SVG 容器尺寸自适应，小球布局自动重新计算
- 视觉反馈：磁吸半径内的小球才有动画反馈，超出范围无交互，交互边界清晰

## 🛠 技术点（Vue3 + GSAP + SVG 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Script Setup 语法糖   | Vue3 推荐写法，代码简洁无冗余，无需手动导出 |
| ref 响应式引用        | 管理 SVG 容器 DOM 元素/磁吸状态参数，状态联动统一 |
| GSAP Timeline 时间轴  | 串联偏移/恢复动画，精准控制动画顺序与延迟，替代原生 setTimeout |
| SVG 元素动态创建      | 使用 createElementNS 动态生成 circle/line 元素，自定义属性配置 |
| 鼠标位置计算          | 基于容器坐标偏移计算相对位置，确保磁吸范围精准匹配 SVG 容器 |
| 距离公式计算          | 勾股定理计算鼠标与小球的距离，判断是否触发磁吸效果 |
| 生命周期管理          | onMounted 初始化/onUnmounted 清理鼠标监听，避免内存泄漏 |
| CSS 定位与布局        | absolute 定位 SVG 容器，flex 居中布局，overflow: visible 展示溢出动画 |
| scoped 样式隔离       | 组件样式独立，避免全局样式污染，SVG 样式精准控制 |
