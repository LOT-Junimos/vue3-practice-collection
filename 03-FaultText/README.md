# FaultText - Vue3 故障文字效果组件
> 一个基于 Vue3 Setup 语法糖实现的故障文字（Glitch Text）效果组件，点击页面触发1秒红蓝偏移+随机裁剪动画，适配移动端/桌面端，支持自定义文字和动画参数。

## 🌐 在线预览
https://vue3-practice-collection-ft.vercel.app/

## 🎯 功能亮点
1. 响应式状态管理：Vue3 ref 单一变量控制故障效果激活状态，状态联动无冗余
2. 随机动画逻辑：每次点击生成不同的偏移/裁剪参数，故障效果更自然
3. 自动重置机制：触发动画后1秒自动恢复初始状态，无需手动干预
4. 样式封装：scoped 样式隔离 + CSS 混合模式，兼容不同项目引入不冲突
5. 响应式适配：针对移动端/平板/桌面端做字体大小自适应，体验一致
6. 内存优化：组件卸载时自动清除定时器，避免内存泄漏

## 📱 效果说明
- 默认状态：黑色全屏背景显示白色「404 NOT FOUND」文字，无故障效果
- 点击页面：文字触发红蓝双影偏移 + 随机位置位移 + 局部裁剪效果，每秒刷新30次随机参数
- 自动恢复：1秒后故障效果停止，文字回到初始位置和样式
- 移动端适配：屏幕越小字体自动缩小，保留故障效果核心视觉

## 🛠 技术点（Vue3 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Script Setup 语法糖   | Vue3 推荐写法，无需 export default/return，代码更简洁 |
| ref 响应式变量        | 控制故障激活状态/文字位置/定时器，状态统一管理 |
| computed 计算属性     | 动态生成文字层列表，灵活控制显示数量 |
| 动态style绑定         | 根据随机参数实时更新 transform/clipPath，实现动态视觉 |
| 点击事件绑定 (@click) | 极简逻辑触发/重置动画，易维护易扩展 |
| scoped 样式           | 组件样式隔离，mix-blend-mode 实现红蓝叠加效果 |
| CSS 响应式布局        | @media 适配不同屏幕尺寸，字体/间距自动调整 |
| onUnmounted 生命周期  | 组件卸载清除定时器，避免内存泄漏 |
