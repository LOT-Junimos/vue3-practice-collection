# PhotoBox - Vue3 交互式照片墙组件
> 一个基于 Vue3 + GSAP 实现的交互式照片墙组件，支持鼠标拖拽移动、缩放反馈、无限循环滚动，适配多屏幕尺寸

## 🌐 在线预览
https://vue3-practice-collection-pb.vercel.app/

## 🎯 功能亮点
1. 拖拽交互核心：鼠标按下/释放/离开多状态联动，拖拽时照片墙跟随鼠标平滑移动
2. GSAP 动画优化：高性能缩放/位移动画，ease 曲线控制过渡更自然
3. 无限循环机制：X/Y轴超出边界自动重置位置，实现无缝循环移动
4. 响应式适配：根据屏幕宽高比自动调整照片尺寸，PC/平板/手机体验一致
5. 细节交互设计：照片 hover 轻微放大，鼠标按下画布微缩，视觉反馈更丰富
6. 内存管理优化：组件卸载时清除 GSAP 动画实例/监听事件，避免内存泄漏
7. 缩放比例适配：基于屏幕宽度动态计算缩放系数，不同分辨率下布局统一

## 📱 效果说明
- 默认状态：黑色全屏背景展示多行列照片墙，居中布局无偏移
- 鼠标按下：画布立即缩小至95%，进入可拖拽状态
- 拖拽移动：照片墙跟随鼠标X/Y轴移动，超出边界自动循环补位
- 释放/离开：画布恢复原缩放比例，拖拽状态终止
- 照片 hover：单张照片放大1.2倍，增强视觉焦点
- 响应式适配：屏幕宽高比变化时，照片尺寸自动调整，保留核心交互逻辑

## 🛠 技术点（Vue3 + GSAP 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Script Setup 语法糖   | Vue3 推荐写法，代码简洁无冗余，无需手动导出 |
| ref 响应式引用        | 管理DOM元素/动画状态/尺寸参数，状态联动统一 |
| GSAP 动画库           | 实现高性能缩放/位移动画，替代原生CSS动画更灵活 |
| 鼠标事件体系          | mousedown/mouseup/mousemove/mouseleave 全流程事件监听 |
| 无限循环算法          | 位移超出边界时重置偏移量，实现无缝循环效果 |
| 动态样式计算          | 基于屏幕宽度计算缩放比例，适配不同分辨率 |
| 生命周期管理          | onMounted 初始化/onUnmounted 清理动画/监听，避免内存泄漏 |
| CSS 响应式布局        | @media 按宽高比调整照片尺寸，rem/em 适配不同屏幕 |
| scoped 样式隔离       | 组件样式独立，mix-blend-mode 实现照片背景融合 |
