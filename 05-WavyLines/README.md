# WavyLines - Vue3 交互式波浪线条组件
> 一个基于 Vue3 + SVG 实现的交互式波浪线条动画组件，鼠标/触摸移动触发网格线条流体波动效果，适配全屏幕响应式，支持自定义动画参数和交互强度。

## 🌐 在线预览
https://vue3-practice-collection-wl.vercel.app/

## 🎯 功能亮点
1. 多端交互支持：同时兼容鼠标移动（桌面端）和触摸滑动（移动端），交互逻辑统一
2. 流体物理动效：模拟鼠标速度/方向计算波动强度，波浪效果随移动速度动态变化
3. 双层次波动：基础正弦波 + 细节波纹叠加，视觉层次更丰富
4. 平滑过渡算法：鼠标位置/速度采用缓动插值，动画无卡顿更自然
5. 高性能渲染：requestAnimationFrame 帧动画 + 按需更新，CPU 占用低
6. 响应式适配：窗口大小变化自动重绘网格，适配任意屏幕尺寸
7. 内存安全：组件卸载自动清除动画帧/定时器，避免内存泄漏
8. 可定制化：支持调整网格间距/波浪强度/交互半径等核心参数

## 📱 效果说明
- 默认状态：浅灰色背景显示黑色网格线条，线条缓慢自然波动
- 鼠标/触摸移动：线条向鼠标方向产生流体偏移，移动速度越快波动越强
- 距离衰减：离鼠标越近的线条偏移越明显，超出交互半径后恢复初始状态
- 弹性回归：停止交互后线条平滑恢复到初始位置，无生硬跳转
- 全屏幕适配：网格自动填充容器，移动端/桌面端视觉效果一致

## 🛠 技术点（Vue3 + SVG 核心）
| 技术点                | 说明                          |
|-----------------------|-------------------------------|
| Options API 经典写法  | 状态/方法分离，动画逻辑更易维护，适合复杂交互场景 |
| ref 模板引用          | 获取 SVG/容器 DOM 元素，精准控制渲染节点 |
| 响应式数据管理        | 统一管理鼠标位置/速度/动画时间等核心状态 |
| SVG 路径绘制          | 动态生成 path 路径，Catmull-Rom 样条曲线实现平滑线条 |
| 事件系统封装          | 封装 resize/mousemove/touchmove 事件，兼容多端交互 |
| 物理动效算法          | 距离计算/速度插值/弹性阻尼，模拟真实流体物理效果 |
| requestAnimationFrame | 浏览器原生帧动画，60fps 流畅渲染无掉帧 |
| 生命周期管理          | mounted 初始化/ beforeUnmount 清理资源，避免内存泄漏 |
| CSS scoped 隔离       | 组件样式独立，mix-blend-mode 兼容不同背景色 |

npm run dev
